# Student Information
## Description
This project is a simple python program that collects and displays student information.
## Technologies used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. Enter the required information.
## Sample output
```text
Enter your name: Ali
Enter your age: 17
Enter your city: Lahore

My name is Ali
I am 17 years old
I live in Lahore
```
